1. PLATFORM OVERVIEW
===============================================================================

GRADIO is a modern, AI-powered quiz creation and management platform designed
specifically for educators. It combines cutting-edge technology with intuitive
design to streamline the entire quiz lifecycle - from creation to analysis.

MISSION: Empower educators with intelligent tools to create engaging assessments
while maintaining academic integrity through advanced monitoring systems.

TARGET USERS:
- K-12 Teachers
- University Professors  
- Corporate Trainers
- Educational Institutions
- Online Course Creators

===============================================================================

FRONTEND FRAMEWORK:

AUTHENTICATION:

4. SECURITY & ANTI-CHEAT SYSTEM
===============================================================================

REAL-TIME MONITORING:
The platform includes a sophisticated anti-cheat system that monitors
student behavior throughout the quiz-taking process.

TRACKED ACTIVITIES:
→ Tab switching detection
→ Window focus loss monitoring
→ Copy/paste attempt logging
→ Right-click context menu blocks
→ Keyboard shortcut interception (F12, Ctrl+C, Ctrl+V, etc.)
→ Fullscreen mode exit detection
→ Time spent away from quiz tab
→ Mouse movement patterns
→ Suspicious activity timestamps

RISK ASSESSMENT ALGORITHM:
→ Dynamic scoring based on multiple factors
→ Tab switches: 10% risk per switch (max 50%)
→ Time away: 5% risk per minute (max 30%)
→ Copy/paste: 15% risk per attempt (max 45%)
→ Right-clicks: 5% risk per attempt (max 20%)
→ Keyboard shortcuts: 10% risk per shortcut (max 40%)
→ Fullscreen exits: 20% risk per exit (max 60%)

RISK LEVELS:
→ Minimal Risk (0-14%): Green indicator
→ Low Risk (15-39%): Orange indicator  
→ Medium Risk (40-69%): Yellow indicator
→ High Risk (70%+): Red indicator

EDUCATOR INSIGHTS:
→ Detailed cheating metrics for each submission
→ Risk score breakdown by activity type
→ Timeline of suspicious events
→ Comparative analysis across students
→ Exportable security reports
